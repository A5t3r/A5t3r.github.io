Hugo website deployment

install hugo

我的环境是Windows, 在官网下载Hugo后, 添加Hugo路径到环境变量, 命令行下输入hugo version, 出现版本信息表示安装成功
$ hugo version
Hugo Static Site Generator v0.55.6-A5D4C82D windows/amd64 BuildDate: 2019-05-18T07:57:00Z

deploy

参考官方教程hugo quick-start, hugo生成静态文件后将public文件中文件push到repo完成部署.

theme

我使用的是Beg, 主题页面有详细的配置说明

hidden page

参考这个答案, 修改quickstart\themes\beg\layouts\_default\list.html文件, 在<ul class="p-articles">处添加
{{ range (.Paginate .Data.Pages).Pages }}
		{{ if not .Params.hidden }}
			<li>{{ .Render "li" }}</li>
		{{ end }}
	{{ end }}

comments

在themes/your_theme/layout/partials添加comment.html, 或者在文章页面添加评论代码, 我使用的是utterances, 优点是授权一个repo就可以使用.
{{ if and .IsPage (ne .Params.comment false) }}
  <!-- utteranc -->
  {{ if .Site.Params.utteranc.enable }}
    <div class="post bg-white">
      <script src="https://utteranc.es/client.js"
            repo= "{{ .Site.Params.utteranc.repo }}"
            issue-term="{{ .Site.Params.utteranc.issueTerm }}"
            theme="{{ default "github-light" .Site.Params.utteranc.theme }}"
            crossorigin="anonymous"
            async>
      </script>
    </div>
	
  {{ end }}
{{- end }}
在single.html(posts page)添加{{ partial "comments.html" . }}, config.toml添加代码
[params.utteranc]
  enable = true
  repo = "github-name/repo-name"
  issueTerm = "pathname"
  theme = "github-light"

favicon(pages icon)

我在主题static目录下存放图标
<link rel="icon" href="{{ "favicon.ico" | relURL }}" >
